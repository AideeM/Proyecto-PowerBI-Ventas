# Proyecto Power BI - Análisis de Ventas

## Objetivo
Analizar tendencias de ventas por producto, región y sucursal usando Power BI.

## Actividades realizadas
- Limpieza de datos
- Creación de modelo
- Visualizaciones
- Análisis de estacionalidad
- Comparación entre regiones

## Herramientas
Power BI Desktop
Power Query
DAX
